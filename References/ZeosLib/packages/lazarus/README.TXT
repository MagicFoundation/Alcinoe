ZeosLib 7.0 Lazarus Edition


Introduction
------------

ZeosLib 7.0 has been tested to work with Lazarus 0.9.24.


Installation
------------

1.  Components / Open Package File (*.lpk)
2.  Open Components.Lpk
4.  Click Install
5.  Click Yes when asked if you want to rebuild Lazarus

That's it!

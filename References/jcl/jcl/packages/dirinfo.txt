JCL package directory; version-specific packages are located in the respective subdirectories.

Files located in jcl/packages:

JclPackagesDxy.bpg:	Borland project group file for Delphi version x.y packages

bcb.gmk:		bpr2mak GNU Make template file for Kylix/C++

Files not intended for end users:

JclDevPackagesDnn.bpg:	Borland project group file for Delphi version nn development packages

With BlowPipeSMS, Send SMS text messages instantly to groups or 
individuals via your internet connected computer. This Software 
can send bulk SMS (Mailing SMS, Alert SMS or Newsletter SMS) using 
normal Mobile phone connected via bluetooth 

The software can be downloaded at: 
http://sourceforge.net/projects/blowpipesms/
